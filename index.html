Long version :

The Preboot eXecution Environment, PXE, allows you to boot a computer from a remote filesystem, instead of a local storage.

It does so by starting a minimal environment that will configure the network interface, fetch a file on the network and boot from it.

The iso image contains a iPXE boot firmare with this script embedded:

#!ipxe
prompt --key 0x02 --timeout 2000 Press Ctrl-B for the iPXE command line... && shell ||
dhcp
sleep 2
chain http://boot.chainboot.com/boot

This will then load pxelinux.0 from http://boot.chainboot.net/, along with a configuration file that contains the menu.

From this menu you can boot the installer of your distribution of choice, straight from the mirrors.

Some useful tools are also in the menu such as the Super Grub2 Disk or the Hardware Detection Tool.

You can boot this preconfigured iPXE image from a usb key or a cd.
