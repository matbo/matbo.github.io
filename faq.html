FAQ

Where are the big files?

Not here, on the distributions official mirrors. Chainboot.net just serves you a pxelinux binary and a config file.

Where does the pxelinux.0 file on this server come from?

Copied straight from the Debian netboot installer. The config file has to be on the same directory as pxelinux.0, that’s why it is here.

Debian is just where the bootloader binary comes from, linux is NOT booted from Chainboot.

This file

  • http://boot.chainboot.net/pxelinux.0

is from

  • http://ftp.debian.org/debian/dists/squeeze/main/installer-i386/current/images/netboot/debian-installer/i386/pxelinux.0

and it should mach the MD5 hash from

  • http://ftp.debian.org/debian/dists/squeeze/main/installer-i386/current/images/MD5SUMS

Be careful, “current” changes from time to time.

What about security?

There is no ssl authentication. But it looks like iPXE supports https.
