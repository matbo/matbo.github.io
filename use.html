Boot from a cdrom

Download the cdrom image:

  • http://www.chainboot.net/chainboot.iso

Burn the iso and reboot on the cd, be careful, burn as an image. You can also test it with qemu:

Boot from an usb key (syslinux)

This example is for debian/ubuntu with a FAT partition on the key. See http://etherboot.org/wiki/syslinux for other methods.

WARNING : for the following, replace sdb1 and sdb with the values for your usb disk. If you do this on your local disk you will loose your current boot loader and even maybe your data, you’ve been warned !

install syslinux if necessary

 sudo apt-get install syslinux

mount the usb disk and copy the iso file on it.

  • http://www.chainboot.net/chainboot.iso

create a syslinux.cfg at the root with :

 DEFAULT chainboot
 LABEL chainboot
   LINUX memdisk
   INITRD chainboot.iso
   APPEND iso

copy memdisk to the usb key, the file is /usr/lib/syslinux/memdisk on debian

umount the disk and install syslinux on the usb key

sudo umount /dev/sdb1
sudo syslinux -s /dev/sdb1
sudo dd if=/usr/lib/syslinux/mbr.bin of=/dev/sdb

Boot from an usb key (windows)

download the cdrom image:

  • http://www.chainboot.net/chainboot.iso

you will need the YUMI Multiboot USB Creator

  • http://www.pendrivelinux.com/downloads/YUMI/YUMI-0.0.4.2.exe

when asked for the distribution to use, scroll down to “Try an Unlisted ISO” and select the chainboot.iso file you downloaded.

Add to a PXELINUX menu

just add this entry to your config file:

LABEL Boot from http://boot.chainboot.net
  kernel http://boot.chainboot.net/chainboot.lkrn
